# mw5-r13vergrrl Research Repository

One-page index of everything in this folder. This research supports building a MechWarrior 5: Mercenaries mod that adds a custom hero pilot, R13verGrrl, with voice lines in Turkish/English, unlock quest, and full integration.

## Files

| File | What's in it |
|---|---|
| **00-README.md** | This index (you are here). |
| **01-implementation-guide.md** | Step-by-step implementation guide for Chris to follow. Covers prerequisites, file structure, DataTable setup, portrait specs, voice wiring, quest unlock system, origin planet, testing, and packaging. |
| **02-voice-lines-script.md** | The recording script Chris hands to the voice actor. Includes English intent + Turkish translation for each line, delivery notes, mood, personality cues, cat meow callouts, and time estimates. |

| **05-open-questions.md** | Things we couldn't determine from research. What Chris will need to figure out or verify at build time. |

## Quick Start

**To build the mod**, read **01-implementation-guide.md** in order. It assumes zero prior MW5 modding experience and is written as a narrative checklist.

**To record voice lines**, read **02-voice-lines-script.md**. It lists every line Chris needs, organized by priority (MUST-HAVE vs NICE-TO-HAVE).

**To track progress**, use **04-tickets-to-create.md** to create GitHub issues.

**To troubleshoot**, check **03-references.md** for the original sources, or **05-open-questions.md** if you hit an unknown.

## Character Summary

- **Name:** R13verGrrl (the girlfriend's nickname in-game)
- **Origin:** Türkiye / Anatolian region of Inner Sphere
- **Personality:** strong-willed, kind, caring, great humor, a bit clumsy
- **Loves:** cars, cats
- **Voice:** Turkish + English mix, with real cat meows in background (she has cats)
- **Unlock:** Complete any single random mission to hire her as a pilot
