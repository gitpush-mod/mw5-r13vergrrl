# Resources/

Mod thumbnail and other non-Content assets.

| File | Purpose |
|------|---------|
| `Icon128.png` | Mod icon shown by the game's mod manager and (optionally) Steam Workshop. 128×128 PNG. Present only once a version exists; can be derived from the portrait. |

## Steam Workshop tile

Workshop's tile image is uploaded through the "Publish to Steam" dialog in the Mod Editor and is **not** stored here — Steam hosts it. See `RESEARCH/01-implementation-guide.md` §8 for the publish flow.
